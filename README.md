# Fastgre
### FastAPI and PostgreSQL integration to simple usage

* ADD
```json
{
   "context":{
      "table_name":"tablename",
      "cols":[
         "col1",
         "col2"
      ],
      "vals":[
         "val1",
         "val2"
      ]
   }
}
```

* DELETE
```json
{
   "context":{
      "table_name":"tablename",
      "condition":"col='val'"
   }
}
```
